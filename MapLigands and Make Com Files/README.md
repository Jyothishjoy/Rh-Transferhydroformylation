# Functionality of `mapligands-com-maker.py` script

`mapligands-com-maker.py` script is used to Gaussian generate input files for Transfer hydroformylation 

The first step is to use AaronTool's `mapLigand` functionality to add PP ligand of our choice to all the template core strutures 
In the second step a geader and footer are added to thus generated XYZ file to prepare the appropriate .com file.
