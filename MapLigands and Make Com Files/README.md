# Functionality of `mapligands-com-maker.py` script

`mapligands-com-maker.py` script is used to generate Gaussian input files for Transfer hydroformylation 

The first step is to use AaronTool's `mapLigand` functionality to add PP ligand of our choice to all the template core structures.

In the second step, a header and footer are added to the generated XYZ file to prepare the appropriate .com file.
